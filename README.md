# \<fizz-buzz>

This webcomponent follows the [open-wc](https://github.com/open-wc/open-wc) recommendation.

## Installation
```bash
npm i fizz-buzz
```

## Usage
```html
<script type="module">
  import 'fizz-buzz/fizz-buzz.js';
</script>

<fizz-buzz></fizz-buzz>
```

## Testing using karma (if applied by author)

```bash
npm run test
```
